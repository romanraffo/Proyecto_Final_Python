EXPLICACION DEL PROYECTO
Este trabajo consiste en un análisis exploratorio de datos (EDA) sobre el dataset TMDB 5000 Movies.
El objetivo principal fue aplicar herramientas de análisis de datos con pandas, generar visualizaciones con matplotlib y seaborn, preparar resultados resumidos y, finalmente, exponerlos mediante una mini-API local desarrollada con Flask.

El proyecto está dividido en dos partes principales:

1. Notebook (.ipynb) donde se realiza:
   -Lectura, limpieza y procesamiento del dataset
   -Transformación de columnas (fechas, géneros, etc.)
   -Estadísticas descriptivas
   -Gráficos
   -Respuestas a los ejes de análisis elegidos
   -Generación de archivos .csv para la API

2. Mini-API local, que lee esos archivos y devuelve los resultados en formato JSON.
